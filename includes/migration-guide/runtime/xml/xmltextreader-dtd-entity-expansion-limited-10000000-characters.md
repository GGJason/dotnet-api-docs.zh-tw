### <a name="xmltextreader-dtd-entity-expansion-is-limited-to-10000000-characters"></a>XmlTextReader DTD 實體展開會限制為 10,000,000 個字元

|   |   |
|---|---|
|詳細資料|DTD 實體展開現在限制為 10,000,000 個字元。 載入無 DTD 實體展開或 DTD 實體展開受限的 XML 檔案並不會受到影響。 若檔案具有展開至超過 10,000,000 個字元的 DTD 實體，則會載入失敗，而且現在會擲回例外狀況。|
|建議|值太低 10,000,000 的 DTD 實體展開限制時，可以覆寫與<xref:System.Xml.XmlReaderSettings.MaxCharactersFromEntities>屬性。 <xref:System.Xml.XmlReaderSettings?displayProperty=name>適當<xref:System.Xml.XmlReaderSettings.MaxCharactersFromEntities?displayProperty=name>值可以傳遞至<code>XmlReader.Create</code>採用<xref:System.Xml.XmlReaderSettings?displayProperty=name>（即<xref:System.Xml.XmlReader.Create(System.String,System.Xml.XmlReaderSettings)>)|
