> [!NOTE]
> **.NET Core 僅在 Linux 和 macOS 系統上執行：** C 和 Posix 文化特性的定序行為一律區分大小寫，因為這些文化特性不會使用預期的 Unicode 定序順序。 建議您使用 C 或 Posix 以外的文化特性來執行區分文化特性、不區分大小寫的排序作業。
